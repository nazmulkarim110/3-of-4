CSS SELECTORS

There are three (3) types of Selectors mostly used in HTML styling are as follows

Element Selector= div, body, section, aside, nav, etc. This selector is not specific, because style on element/tag selector apply on all same element/tag's present in whole html document. Example: p { text-align: center;} Here, all 'p' elements on the page will be center-aligned

Class Selector=The class selector selects HTML elements with a specific class attribute.To select elements with a specific class, write a period (.) character, followed by the class name. Example: .navbar {color:red;}

ID Selector= It uses the id attribute of an HTML element to select a specific element. The id of an element is unique within a page, so the id selector is used to select one unique element! a hash (#) followed by the id of the element is used to select an id. Note: An id name cannot start with a number!Example: #logo {height:auto;}

MEASUREMENT

USE OF COLOR

SOLID:
It is used by giving input of color name directly. Example: color:blue
HEXADECIMAL:
A hexadecimal color is specified with: #RRGGBB, where the RR (red), GG (green) and BB (blue) hexadecimal integers specify the components of the color. Example: #3cb371
RGB:
In HTML, a color can be specified as an RGB value, using this formula:rgb(red, green, blue); For example, rgb(255, 0, 0) is displayed as red
RGBA:
RGBA color values are an extension of RGB color values with an alpha channel - which specifies the opacity for a color. formula: rgba(red, green, blue, alpha); The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (not transparent at all) Example: rgba(17, 141, 134, 0.7)
